COMP 474: Software Engineering
=======================================================

Credit Hours
-----------------------------------

3

Prerequisites
----------------------------

:doc:`comp271`


Description
----------------------------

Using an object-oriented language such as C++ or Java, the student (working in
a small team) will learn to plan, design, implement, and test a large software
project.
